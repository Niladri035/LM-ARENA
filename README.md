# LM Arena - Full-Stack LLM Battlegrounds

LM Arena is a high-performance, full-stack AI evaluation platform that lets you pit models against each other in real-time. Built with a focus on cinematic UX and rigorous technical benchmarks.

## 🚀 Features
- **Dynamic Leaderboards**: Real-time ELO-based ranking of top AI models.
- **Cinematic Battle Arena**: High-end GSAP animations for modern AI evaluation.
- **Persistent History**: MongoDB-backed session persistence for every battle.
- **Cloud Analytics**: Detailed performance metrics including latency, tokens, and cost.

## 🛠️ Stack
- **Frontend**: React, GSAP, Three.js, Lucide-React.
- **Backend**: Node.js, Express, LangGraph, MongoDB.
- **Models**: Mistral, Cohere, Gemini.

## 📦 Getting Started

### 1. Backend Setup
1. `cd Backend`
2. `npm install`
3. Create `.env` with your API keys.
4. `npm run dev`

### 2. Frontend Setup
1. `cd Frontend`
2. `npm install`
3. `npm run dev`
